Welcome to the Simon Says game! Follow these instructions to play:

  • Press any key to start the game.
  • At the start of each round, a new color will flash, adding to the existing sequence.
  • Carefully watch the entire sequence of colors as they light up.
  • Click the colors in the exact same order as the full sequence shown.
  • Each round, the sequence gets longer, making the game more challenging.
  • If you click the wrong color or in the wrong order, the game ends.

Test your memory and reflexes to see how far you can go!
